Capabilities

Three ways to analyze, one rule engine.

Whether you're scanning a deployed target, reviewing an uploaded file, or sanity-checking a code snippet during review, WebGuard runs the same battle-tested rule set across every input.

URL Scanner
Active reconnaissance against live web applications. Probes headers, TLS, CORS, and common injection points without intrusive payloads.
  • Security header analysis
  • TLS/SSL configuration checks
  • CORS policy audit
  • XSS surface fingerprinting
File Upload
Static analysis on source files in seven languages. Detects hardcoded secrets, unsafe function calls, and known-vulnerable patterns.
  • Hardcoded credential detection
  • Injection sink identification
  • Cryptographic weakness scan
  • Dependency CVE matching
Code Review
Paste a code snippet and get instant feedback. Ideal for reviewing isolated functions during PR review or interview prep.
  • SQL injection patterns
  • RCE via eval/exec
  • Unsafe DOM sinks
  • Auto language detection
